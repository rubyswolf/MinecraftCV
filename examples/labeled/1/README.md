This example is taken from the MCC 11 parkour warrior map recreation which can be found at https://drive.google.com/file/d/1MZ34zEBGmcKw-xlI9QdSMHECf70veHC_/view?usp=sharing
You can teleport to the original coordinates with `/tp @s 24.10319081385771, 32.09892840326218, 254.05654024039364 3.1681824, 44.775566`
You can teleport to the recreated coordinates with `/tp @s 24.115220913157 32.101398745033 254.016330783291 3.2763167 44.6224401`
Screenshots were taken with F2 on the default FOV in windowed mode maximized on a 1920x1080 screen with the default windows taskbar and title bar.
You can upload the `annotated.svg` to the full version to load in all of the annotations and test out the PnP/PnL system yourself.
